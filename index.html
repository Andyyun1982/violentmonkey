Violentmonkey

Violentmonkey, provides userscript support for Chromium-based Opera and Chrome.

View on GitHub Download .zip Download .tar.gz

Introduction

Violentmonkey is a userscript manager to support running userscripts in web pages. (Similar to Greasemonkey and Tampermonkey)

Installation

  • Opera: Go
  • Chrome: Go
